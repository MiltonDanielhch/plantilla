# Sistema Semilla 3026

## Misión
Arquitectura de software universal diseñada para liberar el potencial humano.

## Requisitos Previos
- **Rust & Cargo**: [Instalar desde rustup.rs](https://rustup.rs/) (El cerebro)
- **Visual Studio Build Tools**: Requerido en Windows ("Desarrollo para el escritorio con C++").
- **Node.js**: [Instalar LTS](https://nodejs.org/) (La vitrina)
- **Python 3**: Para scripts de automatización.

## Quickstart
1. `python semilla.py` (Scaffolding)
2. `cd backend && cargo run` (Backend)
3. `cd frontend && npm run dev` (Frontend)

## Reglas de Sintonía
- Código limpio y modular.
- Tests antes de deploy.
